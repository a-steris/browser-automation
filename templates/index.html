Getting Started

Follow these steps to set up your Stripe integration:

How to Get Your API Key:

  1. Go to dashboard.stripe.com and login
  2. Switch to Test mode using the toggle at the top right
  3. Go to Developers → API keys
  4. Copy your Secret key (starts with 'sk_test_')

Important:

  • Make sure you're in TEST mode
  • Only use keys that start with 'sk_test_'
  • Never share or commit your API keys
